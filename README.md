## 制作流程

### Phase 1 · 剧本策划

对标《宝可梦礼宾部》确定主题与叙事风格：
- 明确故事主题与情感基调
- 设计叙事节奏、具体分镜、转折点
- 确定每个镜头的时长、内容、运动方式

### Phase 2 · 视觉风格提炼

从参考作品中提取核心视觉元素：
- 色彩体系
- 物品材质
- 光照风格
- 画面特色

→ 沉淀为**基础提示词库**，确保整体风格稳定一致

### Phase 3 · 角色设计

以小熊虫表情包 IP 为原型：
- 构建 3D 羊毛毡风格角色
- 定妆照调试：材质、细节、光影
- 工具：nano banana 2.0

### Phase 4 · 场景设计

- 基于风格提示词 + 角色定妆照生成主要场景
- 多轮风格提示词迭代调试
- 逐步确定场景风格细节

### Phase 5 · 分镜生成

- 依据剧本生成核心分镜图
- 逐镜确认风格一致性
- 为每个分镜图生成对应视频

### Phase 6 · 剪辑 & 配音

- 依据分镜视频完成剪辑
- 简单配音制作
