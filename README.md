## 制作流程

### Phase 1 · 剧本策划

对标《宝可梦礼宾部》确定主题与叙事风格：
- 明确故事主题与情感基调
- 设计叙事节奏、具体分镜、转折点
- 确定每个镜头的时长、内容、运动方式

**具体内容**：
> 在脑中构思对标作品《宝可梦礼宾部》的风格类型，先用 Gemini 确定剧本主题、故事内容、叙事节奏、具体分镜、转折以及整体想要传递的感受，再制定分镜——包括每个镜头持续多久、发生了什么、镜头如何运动等细节。

---

### Phase 2 · 视觉风格提炼

从参考作品中提取核心视觉元素：
- 色彩体系
- 物品材质
- 光照风格
- 画面特色

→ 沉淀为**基础提示词库**，确保整体风格稳定一致

**具体内容**：
> 将对标作品的经典画面上传，与 AI 一起总结画面风格、色彩、物品材质、光照、风格特色等，这些会沉淀为整体提示词基础，以保证风格的稳定性。

---

### Phase 3 · 角色设计

以小熊虫表情包 IP 为原型：
- 构建 3D 羊毛毡风格角色
- 定妆照调试：材质、细节，光影
- 工具：nano banana 2.0

**具体内容**：
> 使用 nano banana 2.0 生成风格化角色的定妆照，过程中不断调试角色材质、细节等——将一个很火的表情包 IP 小熊虫构建成 3D 羊毛毡风格角色。

---

### Phase 4 · 场景设计

- 基于风格提示词 + 角色定妆照生成主要场景
- 多轮风格提示词迭代调试
- 逐步确定场景风格细节

**具体内容**：
> 按照之前生成的风格提示词加上角色定妆照，生成一两个主要场景的分镜。此过程需要不断调试风格提示词以保证生成场景的各种细节符合预期，经历了多轮风格提示词的调试。

---

### Phase 5 · 分镜生成

- 依据剧本生成核心分镜图
- 逐镜确认风格一致性
- 为每个分镜图生成对应视频

**具体内容**：
> 确定风格后，开始生成每个分镜的核心分镜图，再依据分镜图生成每个镜头的视频。

---

### Phase 6 · 剪辑 & 配音

- 依据分镜视频完成剪辑
- 简单配音制作

**具体内容**：
> 完成剪辑后进行简单配音，最后完成整个短片。
